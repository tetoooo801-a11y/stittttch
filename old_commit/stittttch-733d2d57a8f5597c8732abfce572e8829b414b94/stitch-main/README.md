# Stitch Frontend

Next.js frontend for the Stitch luxury salon app.

See the [main README](../README.md) for full setup (backend + MongoDB + environment variables).

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The backend must be running at `NEXT_PUBLIC_API_URL` (default `http://localhost:5000`).
